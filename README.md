# 🌍 智能多语言翻译器

一个功能强大、界面优雅的在线翻译工具，采用苹果风格设计。

## ✨ 主要功能

- 🎤 **语音识别** - 支持多语言语音输入，自动转文字
- 🔄 **实时翻译** - 停止输入后自动翻译，无需点击按钮
- 🗣️ **语音朗读** - 清晰自然的多语言语音播放
- 🌐 **多语言支持** - 支持14种主流语言互译
- 💬 **口语化翻译** - 中文转英文时自动口语化处理
- 📚 **翻译历史** - 自动保存最近翻译记录
- 🎨 **优雅设计** - 简洁的苹果风格界面

## 🌐 支持的语言

- 中文（简体/繁体）
- 英语
- 日语
- 韩语
- 法语
- 德语
- 西班牙语
- 俄语
- 阿拉伯语
- 葡萄牙语
- 意大利语
- 泰语
- 越南语

## 🎯 使用方式

### 文字输入
1. 在左侧输入框输入文字
2. 停止输入1.5秒后自动翻译
3. 翻译结果显示在右侧

### 语音输入
1. 点击"🎤 开始语音"按钮
2. 对着麦克风说话
3. 点击"⏹ 停止录音"
4. 自动将语音转为文字并翻译

### 语音朗读
1. 翻译完成后，点击"🔊 朗读翻译"
2. 可通过滑块调整朗读语速
3. 朗读过程中可随时停止

## 🔧 技术特点

- 纯前端实现，无需后端服务器
- 使用 Web Speech API 实现语音功能
- 集成多个翻译API，确保翻译成功率
- 响应式设计，支持移动设备
